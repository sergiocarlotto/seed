# Modelo De Compilado De Conversa

Use este modelo para pedir ao ChatGPT do celular um compilado estruturado ao
fim de uma conversa desordenada.

```markdown
# Compilado De Conversa

## Contexto

- Projeto:
- Data:
- Origem da conversa:
- Objetivo geral percebido:

## Resumo Executivo

Descreva em poucos paragrafos o que foi discutido e qual parece ser a intencao
principal.

## Ideias E Pontos Levantados

Liste as ideias em bullets, sem decidir ainda se elas sao MVP, futuras,
pendencias ou descartadas.

## Possiveis Demandas

Liste itens que parecem poder virar requisito, tarefa, modulo ou criterio de
aceite.

## Funcionalidades Futuras

Liste ideias que parecem importantes, mas nao precisam entrar agora.

## Pendencias E Perguntas

Liste duvidas, incertezas, informacoes faltantes e pontos que precisam de
decisao humana.

## Decisoes Sugeridas Na Conversa

Liste qualquer decisao que a conversa parece ter sugerido, mas marque como
nao aprovada.

## Riscos Percebidos

Liste riscos de produto, arquitetura, prazo, escopo, operacao ou manutencao.

## Trechos Relevantes

Inclua trechos curtos da conversa que ajudem a preservar contexto.

## Observacoes Finais

Inclua qualquer ressalva sobre incerteza, contradicoes ou pontos que podem ter
sido mal interpretados.
```

## Prompt Sugerido Para O Celular

```txt
Com base em toda esta conversa, gere um arquivo Markdown estruturado usando o
modelo abaixo. Nao trate nenhuma ideia como decisao aprovada. Preserve duvidas,
contradicoes e incertezas. Separe ideias, possiveis demandas, funcionalidades
futuras, pendencias, decisoes sugeridas e riscos. Seja claro o suficiente para outra IA
organizar depois no projeto seed-skill.
```
